# Calculator Application
-------------------------
## Project Overview
A simple C++ console-based calculator that performs:
- Addition
- Subtraction 
- Multiplication
- Division

**Key Features**:
- Menu-driven interface
- Input validation
- Continuous operation until explicit exit
- Error handling for division by zero
